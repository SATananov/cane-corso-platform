USG Certificate Issue Flow Patch

Purpose:
- Add a professional admin-only "Issue USG Certificate" flow.
- Keep Registry, USG Gallery, and USG Certified certificates as three separate layers.
- Let admin choose the exact profile photo that is frozen into the certificate.
- Store an issued certificate snapshot so later profile edits do not silently rewrite the certificate data.
- Add public /certified page as the archive/gallery for issued USG certificates.

What changed:
1. Database/schema
   - New migration: packages/db/drizzle/0008_certificate_snapshot_layer.sql
   - certificates.certificate_image_url
   - certificates.snapshot_json
   - certificates.issued_by_profile_id

2. Admin review
   - Button wording changed to "Издай USG сертификат" / "Issue USG certificate".
   - Each owner photo can issue a certificate with that exact photo.
   - Active certificates open the printable certificate page.

3. Public registry profile for admin
   - Admin-only certificate panel lets admin choose the exact image and issue the USG certificate.
   - Certificate issuing is no longer mixed with Registry approval or USG Gallery selection.

4. Certificate document
   - /certificate/[code] renders a printable A4 USG certificate.
   - Missing data renders as "Няма данни" / "No data".
   - Certificate image comes from the admin-selected certificate image.
   - Verify code and certificate code remain visible.

5. USG Certified archive
   - New page: /certified
   - Shows all active USG certificates issued by admin.
   - Separate from /registry and /gallery.
