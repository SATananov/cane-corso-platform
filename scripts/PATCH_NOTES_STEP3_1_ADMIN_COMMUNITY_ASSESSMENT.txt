Step 3.1 — Admin + Community Assessment

Scope:
- Adds official admin assessment control to /review.
- Keeps admin assessment separate from Registry publication, USG certificate, owner media, and community votes.
- Surfaces public admin assessment on public registry detail.
- Clarifies community rating copy so members understand they can rate published Cane Corso profiles.

Database:
- New migration: packages/db/drizzle/0007_dog_admin_assessments.sql
- New table: dog_admin_assessments
  - one assessment per Cane Corso profile
  - registry decision
  - certificate decision
  - criteria scores 1–5
  - public admin note
  - private admin note

Apply:
1. Copy patch files into the repo root.
2. Run: pnpm db:migrate
3. Run: pnpm workspace:syntax
4. Run: pnpm workspace:verify
5. Run: pnpm --filter @cane-corso-platform/web typecheck
6. Run: pnpm dev

Test flow:
- Open /review as admin.
- Fill “Админ оценка”.
- Save the assessment.
- Open the published public profile under /registry/[slug].
- Confirm the public admin note and criteria scores appear.
- Log in as another member and rate from the community panel on the public profile.

Protected scope:
- Header was not changed.
- Home was not changed.
- /my-dogs was not changed.
