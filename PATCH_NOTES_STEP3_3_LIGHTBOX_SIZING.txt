Step 3.3 — Header-aware image lightbox polish

What changed:
- The image preview/lightbox now measures the sticky header and starts below it.
- The preview is constrained to the visible screen height and no longer opens oversized with vertical scrolling.
- Image display uses contain-fit, so the full photo stays visible inside the viewport.
- The same preview behavior is expanded across the main Cane Corso media surfaces:
  - Admin review media controls
  - Public registry detail
  - Public registry cards
  - USG Gallery
  - Admin members dog cards
  - Owner /my-dogs cards and spotlight
  - Dog profile edit/create preview photos
  - Pedigree editor ancestor photos

No database migration is required for this patch.
