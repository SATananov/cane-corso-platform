# Media governance — Owner profile, Registry and USG Gallery

This document locks the product rule for Cane Corso media.

## 1. Owner profile media

Owner-uploaded Cane Corso photos are personal profile media.

Rules:

- The owner can upload 1 to 3 main Cane Corso profile photos.
- These photos are saved immediately when the owner saves the draft/profile.
- The owner must see them in `/my-dogs` without waiting for admin approval.
- Pedigree photos are separate: mother, father and deeper ancestors each use their own pedigree photo.

## 2. Public Registry media

The Registry is an official public layer.

Rules:

- Owner photos do not become publicly relevant until the Cane Corso profile is approved and published by admin.
- A public Registry entry can show the main photo and up to 3 profile photos.
- Publication to the Registry does not automatically mean a USG certificate.
- Registry visibility and certificate trust are separate product decisions.

## 3. USG Gallery media

The USG Gallery is an elite curated layer.

Rules:

- USG Gallery must not automatically include all owner-uploaded photos.
- Only admin-selected photos may enter the USG Gallery.
- The Gallery should represent the best, strongest and most premium Cane Corso images.
- Future admin controls should allow: make primary, show/hide in Registry, add/remove from USG Gallery.

## Locked product separation

Owner upload -> visible to owner immediately.

Admin publish -> visible in public Registry.

Admin curate -> selected images enter USG Gallery.
