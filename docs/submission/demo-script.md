
# Demo Script — Cane Corso Platform

## 1. Public entry

Open `/` and explain that the platform is a Cane Corso ecosystem: Registry, Verify, Knowledge, Community, Partners, and owner/member workspace.

## 2. Member flow

Login as member, open `/my-dogs`, create or edit a Cane Corso profile, save draft, review the owner happy-path panel, and submit the profile for review.

## 3. Admin moderation

Login as admin, open `/review`, show the moderation action-flow panel, review submitted profile data, save assessment, approve or request changes.

## 4. Registry release

From admin flow, publish the approved profile. Open `/registry` and show the public Registry record and release-flow panel.

## 5. Certificate and Verify

Issue a USG certificate, open `/verify`, paste or open the certificate code, and explain that Verify confirms active certificate trust only.

## 6. Architecture Q&A

Open `docs/architecture/nextjs-rendering-data-cache-map.md` and explain Server Components, Client Components, Route Handlers, data fetching, cache boundaries, and performance decisions.
